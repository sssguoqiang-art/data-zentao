# Data Zentao Claude Code 工作指引

本项目用于让 Claude Code 通过自然语言查询禅道数据库，并生成面向项目推进和管理分析的中文总结。

请遵守 `AGENTS.md` 中的工作流程和操作约束。核心原则：

- 先查真实数据，再总结。
- 优先使用 `data-zentao` 命令。
- 自由问题可以通过 `schema + query` 组合完成。
- 只读，不写入。
- 输出要让非技术用户一眼看懂。
- 不要把命令输出原样作为最终答案；必须基于查询结果做判断、归纳、风险解释和下一步建议。
- 区分 `bug-boundary` 和 `version-review`：前者是复盘前 Bug界定预分类，后者是正式版本复盘材料。
